# Student Timetable App

A simple and intuitive Android application built with Jetpack Compose to help students manage their daily class schedules.

## Features

- **Daily Schedule View:** Quickly switch between weekdays (Monday to Friday) to see your planned activities.
- **Dynamic Updates:** Update your schedule for any day in real-time using the built-in input field.
- **Modern UI:** Clean and responsive interface designed with Material Design 3.
- **State Management:** Uses Jetpack Compose's `remember` and `mutableStateMapOf` to keep your schedule updated as you interact with the app.

## Getting Started

### Prerequisites

- Android Studio Flamingo or newer.
- Android SDK level 33 or higher.
- Kotlin 1.8.0 or newer.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/student-timetable.git
   ```
2. Open the project in Android Studio.
3. Wait for Gradle sync to complete.
4. Run the app on an emulator or a physical device.

## Technologies Used

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Theme:** Material Design 3
